# RDI 血管分割

本目录旨在对 RDI（Retinal Digital Imaging）图像中的血管进行分割，重点关注红色血管、蓝色血管以及粗细血管的分割。  

## 结论

- **红色血管分割**  
  由于红色血管结构复杂，传统图像处理方法难以实现可靠分割。当前阶段**仅能通过深度学习方法**实现较准确分割，但尚未完全实现。

- **蓝色粗血管分割**
- 使用红色通道进行处理,RDI图像中蓝色血管在红色通道突出
  蓝色血管结构较为规则，**传统图像处理方法可以实现较好分割**。  

## 参考

- 复杂血管分割目前最好的办法依然是深度学习
- 红色血管分割复杂度高，尚无现成传统方法可用。
